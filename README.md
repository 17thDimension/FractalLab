Fractal Lab
============

An interactive WebGL based fractal raytracer.
See it online at [fractal.io](http://fractal.io).

**Compatibility**

Works best on Chrome 9 but also tested on Webkit nightly, Firefox 4.0b9+ on Max OS X.
